Caves of Qud 日本語化

注意: 原文（英文）からの日本語訳のほぼ全ては生成AIによる翻訳です。一応目を通してはいますが、文脈による意図解釈のズレや誤訳、あるいは珍訳が混じっている可能性が非常に高いです。その場合にはご一報いただけると助かります。それと公式によるローカリゼーションがすでに始まっているので、こちらのプロジェクトはすぐに畳むかもしれません。

概要
Caves of Qud を日本語で遊ぶための非公式ファン制作 Mod です。UI、会話、能力・変異説明、ポップアップ、メッセージログ、一部の自動生成テキストを日本語化し、日本語表示用の CJK フォントを同梱しています。

対応バージョン

  * Caves of Qud 2.0.4

主な内容

  * 会話テキストの日本語化
  * メニュー、オプション、キャラクター作成、能力画面などの UI 日本語化
  * ポップアップ、メッセージログ、ジャーナル、死亡表示などの日本語化
  * 一部の自動生成テキスト、履歴文、説明文の日本語化
  * 日本語表示用 CJK フォントの同梱

既存セーブについて

既存セーブでも基本的に使用できます。ただし、重要なセーブデータは導入前にバックアップすることをおすすめします。セーブ内にすでに保存されている生成済みテキストや、一部の動的テキストは英語のまま残る場合があります。

導入方法

  * サブスクライブ
  * ゲーム内の Mod Manager で Caves of Qud 日本語化 を有効化
  * ゲームを再起動。以上！

報告・Contribution

このプロジェクトは GitHub で公開しています。何か問題があれば、この Mod のコメント欄に投稿していただくか、GitHub Issue として報告してください。Issue の場合は Player.log も添付してもらえると助かります。

  * GitHub: https://github.com/ToaruPen/coq-japanese_stable
  * macOS Player.log: ~/Library/Logs/Freehold Games/CavesOfQud/Player.log
  * Windows Player.log: %USERPROFILE%\AppData\LocalLow\Freehold Games\CavesOfQud\Player.log

Contribution も待っています！

注意事項
  * これは Freehold Games 公式の日本語化ではありません。
  * ゲーム本体のアップデートにより、一部表示やパッチが壊れる可能性があります。
  * 未翻訳・表示崩れ・不自然な訳が残っている場合があります。

未対応・観測している問題

  * プロシージャル全般
  * チュートリアル
  * キャラクター生成画面
  * メッセージログに表示される文字に付与される色タグのズレ
  * その他、細かい UI の表示や観測漏れなど
